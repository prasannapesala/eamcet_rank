# EAMCET College Rank Predictor

A web-based PHP application that helps students estimate college admission possibilities based on their EAMCET rank.

## Database Setup

Import the `colleges.sql` file into MySQL before running the project.

## Features

* Search colleges using EAMCET rank
* Display college information and cutoff details
* Simple and user-friendly interface
* Built using PHP, HTML, CSS, and MySQL

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## Installation

1. Install XAMPP.
2. Copy the project folder to `htdocs`.
3. Start Apache and MySQL.
4. Import the database into MySQL.
5. Open the project in your browser using:
   `http://localhost/project-name`

## Project Structure

* PHP files for backend logic
* CSS files for styling
* Image files for college logos and assets
* MySQL database for storing college data
